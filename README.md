# Recommendation System

這個 Repo 會放我學習推薦系統的筆記，以及一些實作的程式碼。

# Notes

**推薦系統的目標**：預測使用者的偏好或行為，並根據這些預測結果，推薦給使用者可能感興趣的物品。

## 常見的推薦系統種類

### Content-based Filtering
根據商品屬性的相似度，推薦給使用者相似的商品。
舉例來說，如果使用者喜歡看恐怖片，那麼推薦系統就會推薦給使用者其他的恐怖片。
